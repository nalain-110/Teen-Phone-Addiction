🔬 Research Project

ABOUT THE PROJECT

A machine learning system designed to detect and predict smartphone addiction levels in teenagers

🎯
Project Goal
To develop an AI-powered tool that predicts daily phone usage hours in teenagers based on behavioral, psychological, and social factors — enabling early intervention for at-risk youth before addiction becomes severe.
📊
Dataset
The model is trained on a comprehensive dataset of teen behavioral patterns including sleep habits, academic performance, anxiety levels, social media usage, gaming time, family dynamics, and psychological indicators.
🤖
ML Algorithm
K-Nearest Neighbors (KNN) Regression with k=2 neighbors is used to predict daily phone usage hours. The model learns from similar teen profiles to make accurate predictions on new, unseen data.
🚨
Impact
Early detection of phone addiction can help parents, educators, and counselors take timely action. Our risk classification system provides actionable advice tailored to the severity level detected.
